Mathologica

What is Mathologica?

Mathologica is a symbolic math evaluation engine. The Mathologica evaluation engine can understand the math users type. Therefore, it can solve problems flawlessly and provide step by step work in its solve method. Furthermore, it is able to determine the type of problem the user has typed in and the potential ways to solve the problem. In math, there are often multiple methods to solving a problem or multiple ways to manipulate an equation. Mathologica allows the user to account for all of these possibilities with the evaluation drop down selection. Mathologica supports math from basic algebra to advanced calculus.
