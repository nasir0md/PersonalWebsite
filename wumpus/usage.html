Usage

Wumpus Engine

To use this project you must have Visual Studio 2012 installed and the XNA 4.0 redistributable installed. The project does NOT come with the required XNA libraries. Once you have all of this installed download the project and you can open it with Visual Studio.

Controls

Movement Controls: (all modes)

  • W -> Move forward
  • A -> Move left
  • S -> Move backward
  • D -> Move right
  • R-Click -> Change look orientation by dragging around mouse
  • L-Click -> Fire

Player Controls: (play mode only)

  • X -> Interact with map
  • Space -> Jump
  • E -> Toggle weapon raised
  • R -> Reload
  • B -> Drop holding
  • N -> Interact with object
  • H -> Call orblet (Orblets are apart of the Hunt the Wumpus game itself, you can only call them when the Hunt the Wumpus Game is set up)
  • F -> Flashlight
  • L-ALT+F -> To edit mode

Form Creation: (edit mode only)

  • L-CTRL+Shift+S -> Save as scene
  • L-CTRL+S -> Save scene
  • L-CTRL+L -> Load scene
  • L-CTRL+W -> Light editor
  • L-CTRL+O -> Game object editor
  • L-CTRL+A -> Ambient occlusion settings
  • L-CTRL+T -> Shadow settings
  • L-CTRL+P -> Particle settings
  • L-CTRL+G -> Renderer settings
  • L-CTRL+N -> AI editor
  • L-CTRL+E -> Scene settings
  • L-CTRL+U -> Event editor

Misc Commands: (edit mode only)

  • L-ALT+1 -> Display final render target
  • L-ALT+2 -> Display depth render target
  • L-ALT+3 -> Display normal render target
  • L-ALT+4 -> Display light render target
  • L-ALT+5 -> Display specular light render target
  • L-ALT+6 -> Display ambient occlusion render target
  • L-ALT+F -> To edit mode
  • L-ALT+G -> To play mode
  • L-ALT+R -> Kill all of the AI in the scene
  • L-ALT+Q -> Move faster
  • L-ALT+E -> Move slower
  • M-CLICK -> Select object
